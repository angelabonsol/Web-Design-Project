In this section, we had learned multiple graphic features that can be done with our website including:

  • Web Photo Album

    There are different types of picture formats, where one format must be used for a specific purpose.

    • JPG- used for still images and photography
    • GIF- used for simple animations and graphics with flat colors and without gradients.
    • PNG-8 - used for better transparency but no animation and great for icons.
    • PNG-24 -Handles still images and transparency.
    • SVG- used for graphics, logos for web, retina and high-dpi screens.
    It is better to always add an "alt=" feature to caption your images, even for the simplest things.

  • Button

    For a more complex button, it is best to layer them. When making buttons, it is better to give more space when editing them. "" is the format used to make an icon.

  • Web Page Banner

    Layering pictures is the key to making a banner. Specific selection tools are used. Blending pictures together to make it more smooth. Choosing a specific banner color background is also a must so that it blends in with the website.
